# 🎨 Art Source Repository
피겨 스케이팅 프로젝트의 3D 모델링 및 텍스처 원본 저장소입니다.

## 📂 폴더 구조
- `/Characters`: 플레이어 및 마스코트 .blend 파일
- `/Costumes`: 의상 시뮬레이션용 .blend 파일
- `/Textures`: Substance Painter 원본 및 PSD

## ⚠️ 주의사항
- FBX Export 경로는 반드시 팀 공용 드라이브 혹은 로컬 UE5 폴더로 지정하세요.
- 커밋 메시지에 [Char], [Env] 태그를 붙여주세요.
